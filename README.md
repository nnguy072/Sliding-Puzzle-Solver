# Eight Puzzle Solver
CS170: 8 Puzzle Solver (Well, I think it can do 15/25/etc puzzle too)
---
To run: 
1. "make" to compile
2. "make run" to run: 
    1 for default 3x3 puzzle 
    2 for custom puzzle
---
Can solve using:
- [x] Uniform Cost Search
- [x] A\* with Misplaced Title Heuristic
- [x] A\* with Manhattan Distance Heuristic
